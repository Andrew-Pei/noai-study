# 重庆八中 NOAI 竞赛学习平台

## 项目简介

专为重庆八中学生设计的 NOAI 2026 人工智能竞赛备考学习平台，紧扣考试大纲，提供系统化的 AI 学习路径，帮助学生高效备赛。

## 竞赛大纲

根据 NOAI 2026 官方大纲，考试内容分为三大模块：

- **人工智能基础 (30%)** - Python编程、机器学习基础、深度学习基础、AI伦理法律
- **深度学习应用 (40%)** - CNN、RNN、生成模型、深度强化学习
- **AI+ 系统创新 (30%)** - RPA、AI智能体、提示工程与LLM、智能体系统设计

## 网站功能

### 学习模块
1. **AI基础概念** - 人工智能历史、发展、应用领域
2. **Python编程** - 语法基础、数据类型、控制结构、函数模块
3. **机器学习基础** - 监督学习、无监督学习、scikit-learn
4. **神经网络基础** - 神经元原理、网络结构、PyTorch入门
5. **CNN与RNN** - 图像识别、序列数据处理
6. **生成模型** - GAN、VAE、扩散模型
7. **强化学习** - 深度强化学习基础
8. **LLM与提示工程** - 大语言模型原理、Prompt技巧
9. **AI智能体系统** - Agent设计、RPA应用

### 备考工具
- **周计划学习** - 6周渐进式学习路径
- **每周测验** - 通过测验解锁下周内容
- **练习题库** - 历年真题（2024-2025）、专项练习、模拟考试
- **学习进度追踪** - 可视化展示学习成果
- **学习资源** - 推荐书籍、课程、工具

## 技术栈

- **前端框架**: Bootstrap 5.3
- **图标库**: Bootstrap Icons
- **字体**: Google Fonts (Noto Sans SC, Roboto)
- **交互**: 原生 JavaScript
- **设计**: CSS3 渐变、阴影、动画效果

## 快速开始

1. 克隆仓库到本地
2. 双击 `index.html` 在浏览器中打开
3. 开始学习之旅

## 学习路径建议

- **第3周**: 神经网络与深度学习 → 每周测验
- **第4周**: CNN与RNN → 每周测验
- **第5周**: 生成模型与强化学习 → 每周测验
- **第6周**: LLM与提示工程 → 每周测验
- **最后冲刺**: 模拟考试、错题复习
