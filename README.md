# AI学习中心 - 中学生人工智能学习网站

## 项目简介

这是一个专为中学生设计的人工智能学习网站，旨在用简单有趣的方式普及AI知识。网站包含完整的AI学习路径，从基础概念到实践应用，适合12-18岁的中学生学习。

## 网站特点

### 🎯 目标用户
- 中学生（12-18岁）
- 对AI感兴趣的初学者
- 希望了解AI基础知识的教师和家长

### 🚀 核心功能
1. **AI基础知识** - 人工智能概念、历史、应用
2. **机器学习入门** - 监督学习、无监督学习、强化学习
3. **神经网络基础** - 神经元、网络结构、深度学习
4. **AI伦理与未来** - 社会影响、伦理问题、发展趋势
5. **互动学习体验** - 图像识别、聊天机器人、推荐系统演示
6. **学习资源推荐** - 书籍、课程、工具、实践项目

### 🎨 设计特色
- 现代化UI设计，使用Bootstrap 5框架
- 响应式布局，适配手机、平板、电脑
- 色彩鲜明，适合青少年审美
- 大量动画和交互效果增强学习体验
- 简洁易懂的语言和示例

## 技术栈

- **前端框架**: Bootstrap 5
- **图标库**: Bootstrap Icons
- **字体**: Google Fonts (Noto Sans SC, Roboto)
- **图表**: Chart.js (机器学习演示)
- **交互**: 原生JavaScript
- **设计**: CSS3动画、渐变、阴影效果

## 使用方法

### 快速开始
1. 双击 `start.html` 文件在浏览器中打开
2. 点击"进入网站"按钮
3. 开始探索AI学习之旅

### 学习路径建议
1. **初学者**: 首页 → AI基础 → 互动学习
2. **进阶学习**: AI基础 → 机器学习 → 神经网络
3. **实践应用**: 互动学习 → 学习资源 → 实践项目
4. **深度思考**: AI伦理 → 讨论与反思

### 互动演示说明
1. **图像识别演示**: 上传图片或选择示例，体验AI识别物体
2. **聊天机器人**: 与AI助手对话，了解自然语言处理
3. **推荐系统**: 选择电影类型，体验个性化推荐
4. **线性回归演示**: 调整参数，观察机器学习过程
